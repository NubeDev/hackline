# hackline UI

Single React + TypeScript admin UI for `hackline-gateway`. Lives at
[`hackline-ui/`](./hackline-ui/) and is the long-form replacement for
the tiny vanilla-JS bundle currently served from
`crates/hackline-gateway/static/admin/`.

See [`hackline-ui/README.md`](./hackline-ui/README.md) for setup and
boundary rules.
